## Problem

DocuIntelli requires reconciling outputs across multiple frameworks (IFRS vs internal model, regulatory vs operational) to identify conflicts and coverage gaps.

## Proposed solution

Implement reconciliation:
- Input: two or more framework outputs (dimensions + findings + citations).
- Output: reconciliation matrix highlighting overlaps, conflicts, and missing coverage.
- Store as artifacts and render in UI (table + filters + citations).
