# REG-003 — RegSygnal UI: regulatory dashboard, diffs, tasks, and decision linking

- Type: **story**
- Milestone: `M10-REGSYGNAL`
- Repo alignment: **missing**
- Labels: `frontend`, `domain:regsygnal`
- Depends on: `REG-002`, `STUDIO-003`

## Problem

Need UX to view changes, impacted controls, and create tasks/decisions.

## Proposed solution

- Dashboard for latest updates and impacted areas.
- Diff viewer and evidence drilldown.
- Ability to create decisions and tasks linked to evidence.

## Acceptance criteria

- Users can browse updates, see diffs, and create decisions/tasks with citations.

## Test plan

- E2E: ingest update → view diff → create decision.
